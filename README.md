# FFR120---Spreading-Dynamics-in-Brain-Networks-J2.2
Propagation of Alzheimer’s disease on a structural connectivity brain network with and without the influence of amyloid-β deposition. This was done in course FFR120 - Simulation of complex systems by group J2.2.

The code used in the project can be found in the file SI_model_groupJ2.2.ipynb. The code does not include the plots made for the figures in the report.
